4 - Buttons #permalink

We set a base level of styling across all core html elements. This is to remain simplistic and anything requiring more advanced styling will require classes.

4.1 - Buttons - Basic #permalink

Button styles can be placed on any element with the .button class. However it is best if it is an [a] for links to other pages / sites or a [button] if it's for functionality. The default button styles should be left alone if possible. This will leave a clean slate if a dramatically new button style needs applying.

  • Default styling
  • :visited - Shows the user links that have been visited
  • :focus - For tabbing and touch enabled devices
  • :hover - For mouseovers
  • :active - When the button is clicked / pressed
