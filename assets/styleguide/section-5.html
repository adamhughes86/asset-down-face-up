5 - Forms #permalink

Styling forms should be done with care and consideration and maintain usability. It is recommended to use classes to override core styles and not style the elements themselves.

5.1 - Input Field #permalink

Input box and Label styling

  • Default styling
  • .form-field--alt - Alternate style to make input stand out more
Default
.form-field--alt 
<div class="form-field [modifier class]">
  <label class="form-label" for="login-example">Password:</label>
  <input id="login-example" type="text" name="login[example]" class="form-input" />
</div>

5.2 - Textarea Field #permalink

Textarea box and Label styling

  • Default styling
  • .form-field--alt - Alternate style to make input stand out more
Default
.form-field--alt 
<div class="form-field [modifier class]">
  <label class="form-label" for="textarea-example">Textarea</label>
  <textarea type="text" name="textarea" class="form-textarea" rows="2" cols="80" maxlength="200"></textarea>
</div>

5.3 - Input Select #permalink

Select box

Default 
<div class="form-field--select [modifier class]">
  <label class="form-label" for="select-example">Select</label>
  <select id="select-example" type="text" name="select[example]" class="form-select">
    <option value="1" selected="selected">Option 1</option>
    <option value="2">Option 2</option>
  </select>
</div>
