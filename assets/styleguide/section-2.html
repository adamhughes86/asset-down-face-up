2.7 - Address #permalink

The address tag is not used for general postal address unless that address directly relates to the document. The address tag should be used for providing contact info for document maintainers. This can include a feedback address (email or postal), phone numbers or links to external web pages. As a side note, if you need to format an address that doesn't fall into this criteria use HTML5 microdata.

Author: Adam Hughes @ Rippleffect
Default 
<address>
  Author: <a href="mailto:#">Adam Hughes</a> @ Rippleffect
</address>
